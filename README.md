# Namal Portfolio React

React conversion of Namal Chamodya's portfolio with GSAP, ScrollTrigger, and Three.js background.

## Scripts

- `npm start` — run dev server
- `npm run build` — production build
- `npm run preview` — serve build locally

## Assets

Put your images (dp3.png, campus.jpg, projece2.png) in `public/`.

## Notes

- Font Awesome is loaded via CDN in `public/index.html`.
- GSAP animations are initialized in `src/utils/gsapSetup.js`.
- The 3D background is a simple torus knot for visual motion; tweak geometry/material for your vibe.
